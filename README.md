## About 
This project demonstrates the best practises to be followed when developing a spring boot project. 

## Pre-requisites
- Docker
- Docker compose
- Java 12 (openjdk)
- Maven 3

## How to run
Build the jar using command,
    
    mvn clean install -DskipTests

Start the application with the command,
    
    docker-compose up
